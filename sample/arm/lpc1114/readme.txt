哲学者の食事の問題


<動かし方>
  arm-none-eab-gcc などのツールのある環境で

  hos-v4a/sample/arm/lpc1114/gcc

  にて make

  出来上がった hex ファイルを Flash Magic などで焼く


  UART に 115200bps 8bit ノンパリティー で結果を出力。


  DTR/RTS 信号を開放しないと起動しないので注意。
  TeraTerm の場合、以下のマクロを実行後にリセットボタン

  setdtr 0
  setrts 0
